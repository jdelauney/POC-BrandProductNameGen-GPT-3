# Name generator powered by OpenAI - GPT-3

## Description

POC of a brand, business, product, application name generator powered by AI with OpenAPI GPT-3 model

## Dependencies
- vite
- typescript
- sass-swing

## Installation
```bash
> pnpm install
```
## Launch
```bash
> pnpm start
```
## Build
```bash
> pnpm build
```
